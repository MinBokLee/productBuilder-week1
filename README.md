# 2026 신 로또 번호 생성기

AI 통계 기반 추천을 제공하는 로또 번호 생성기입니다.

## 주요 기능

*   **AI 통계 기반 번호 추천:** 과거 당첨 데이터를 기반으로 가중치를 적용하여 번호를 추천합니다.
*   **5게임 동시 생성:** 한 번의 클릭으로 5개의 로또 번호 조합을 생성합니다.
*   **보너스 번호:** 보너스 번호를 포함하거나 제외할 수 있는 옵션을 제공합니다.
*   **결과 복사:** 생성된 번호를 클립보드에 쉽게 복사할 수 있습니다.
*   **테마 변경:** 라이트 모드와 다크 모드를 지원합니다.

## 사용 방법

1.  **번호 생성:** '5게임 추천받기' 버튼을 클릭하여 추천 번호를 생성합니다.
2.  **보너스 번호 설정:** '보너스 번호 포함' 스위치를 사용하여 보너스 번호 포함 여부를 선택합니다.
3.  **결과 복사:** '결과 복사' 버튼을 클릭하여 생성된 번호를 복사합니다.

## 주의사항

*   본 시스템은 확률적 가중치를 사용하지만, 실제 당첨을 보장하지 않습니다.
*   로또는 매 회차 독립적인 확률로 시행되는 무작위 추첨 게임입니다.
*   소액으로 즐겁게 이용하시기 바랍니다.
